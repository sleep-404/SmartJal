# Hackathon Provided Data

Data provided by the Ground Water Department via SmartJal.zip

## Contents

| Folder | Description |
|--------|-------------|
| `Aquifers_Krishna/` | Aquifer boundaries shapefile (8 aquifer types) |
| `GM_Krishna/` | Geomorphology shapefile |
| `GTWells_Krishna/` | Bore well registry (88,988 wells) |
| `LULC_Krishna/` | Land Use Land Cover shapefile |
| `WaterLevels_Krishna/` | Piezometer water level readings (138 stations, 28 years) |
| `UseCase_extracted/` | Villages, Mandals, Soils shapefiles |
| `SmartJal_extracted/` | Additional extracted data |
| `Pumping Data.xlsx` | Village-level pumping/extraction estimates |
