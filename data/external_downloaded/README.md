# Externally Downloaded Data

Data downloaded from public sources to supplement hackathon data.

## Contents

| Folder | Source | Description |
|--------|--------|-------------|
| `dem/` | SRTM/Copernicus | Digital Elevation Model (30m resolution) |
| `grace/` | NASA GRACE | Groundwater storage anomaly time series |
| `rainfall/` | IMD/CHIRPS | Monthly rainfall rasters |
| `satellite/` | Various | Remote sensing products |
| `landcover/` | ESA WorldCover | Land cover classification (10m) |
| `cgwb/` | CGWB Portal | Additional water level data |

## Satellite Data Details

- **MODIS ET** - Evapotranspiration (8-day composite)
- **Sentinel NDVI** - Vegetation index
- **SMAP** - Soil moisture
